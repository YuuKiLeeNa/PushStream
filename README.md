这是一个推流客户端,实现了推送RTMP和RTSP流,DXGI采集桌面，采集摄像头，实现了文字、图片、摄像头水印，枚举采集声音的输入和输出设备,降噪音去除回声环境噪音,采集到的各种设备声音的同步,混音,录制等功能.This is a streaming client that supports pushing RTMP and RTSP streams, DXGI desktop capture, camera capture, and implements text, image, and camera watermarks. It enumerates audio input and output devices, reduces noise (including echo and environmental noise), synchronizes audio from various devices, mixes audio, and records audio.
